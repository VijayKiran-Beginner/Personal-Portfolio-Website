## ✨ Features

- Responsive design — works across desktop, tablet, and mobile
- Animated hero section with a dynamic typing effect cycling through roles
- Sticky navbar with scroll-based styling and mobile hamburger menu
- About section with animated skill progress bars (triggered on scroll into view)
- Projects showcase with hover overlays linking to live demos and GitHub repos
- Contact form with real-time client-side validation
- Resume download button (forces PDF download via Fetch API)
- Back-to-top button with smooth scroll
- Auto-updating footer year

## 🔄 Pushing Updates to GitHub

Made changes to the portfolio? Run these from inside the portfolio folder:

```bash
# Step 1 — go into the portfolio folder
cd Desktop/ApexPlanet/portfolio

# Step 2 — check what changed
git status

# Step 3 — stage all changes
git add .

# Step 4 — commit with a clear message
git commit -m "feat: add new project / fix: update bio / style: adjust hero"

# Step 5 — push (GitHub Pages updates in ~60 seconds)
git push
```

> First time pushing from this machine? Run `git push -u origin main` instead of `git push` in step 5. After that, `git push` alone works every time.
